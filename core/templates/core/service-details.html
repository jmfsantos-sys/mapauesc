Escopo do Mapa Aprimorado

O mapa tem como objetivo fornecer informações completas sobre os locais da UESC, permitindo visualizar rotas e a localização atual do usuário, sendo útil tanto para visitantes quanto para quem se encontra perdido no campus.

Descrição

  • Uso da mesma imagem estática do Mapa da UESC
  • Localização de auditórios (ex.: auditório Paulo Souto)
  • Informações detalhadas por andar dos prédios (torres e pavilhões)
  • Numeração das salas em cada andar
  • Barra lateral com nomes dos locais, escondida até interação
  • Interação com o mapa via clique/hover mostrando modais
  • Imagens internas dos prédios com ícones de informações
  • Localização atual e em tempo real do usuário
  • Rota de origem-destino, com default na guarita
  • Área de contato para sugestões e atualizações
  • Modificação das imagens de locais alterados, como o campo perto da guarita

Backlog e Sprints

O projeto está organizado em quatro sprints, contemplando desenvolvimento Angular, backend em Spring Boot e integração fullstack com autenticação e responsividade.

Sprint 1 - Protótipo Angular (12/09/2025) - 15%

  • Criar Mockup no Figma
  • Criar estrutura de layout
  • Fazer marcações dos locais no mapa UESC
  • Criar rota inicial para o mapa
  • Inserir API do mapa (Leaflet/OpenLayers/Google Maps)
  • Procurar informações para compor o modal

Sprint 2 - Angular Avançado (30/09/2025) - 20%

  • Criação do modelo de entidade e relacionamento (MER)
  • Criação de dados mockados
  • Integrar biblioteca de mapas
  • Implementar interações com o mapa
  • Trajetória (rotas origem-destino)
  • Localização atual em tempo real

Sprint 3 - Backend Spring Boot (22/10/2025) - 25%

  • Modelagem do banco de dados
  • Criar projeto Spring Boot para gerenciar pontos do mapa
  • Substituir dados mockados por dados reais
  • Teste de integração

Sprint 4 - Fullstack + Auth (24/11/2025) - 25%

  • Listagem e pesquisa de nomes dos locais
  • Janela modal com informações
  • Adicionar links dos locais
  • Link de contato
  • Ajustar design
  • Implementar filtragem e identificação de eventos
  • Teste final
  • Responsividade para celular
